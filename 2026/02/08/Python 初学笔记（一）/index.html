Python 初学笔记（一）

发表于 2026-02-08

程序设计与Python环境

程序设计与算法的概念

程序设计发展历程

  1. 机器语言：二进制，不用翻译解释。执行速度快。

  2. 汇编语言：使用注记符号来表示机器语言，需要翻译成机器语言后才能执行。

  3. 高级语言：接近自然语言，使用与自然语言语法接近的自封闭词法体系。

    • 语言处理系统：负责把高级语言的源程序转化为机器语言的软件。同一种高级语言在面对不同机型时应该使用不同的语言处理系统。

    • 高级语言的编译和解释：

      • 编译型：整体转化、集中翻译、最后执行
      • 解释型：对源程序代码一边翻译一边执行

  4. 高级语言体系简介

    1. 以早期BASIC为代表的流水账式程序：已出现基本的分支和循环结构，但尚未有函数的概念。
    2. 以Pascal和C语言为代表的结构化程序设计：出现函数的概念——人们把能够完成一个特定功能的代码块称为函数（或过程）。
    3. 以C++为代表的面向对象的程序设计：出现对象的概念——对象是对即将开发的应用系统中特定物体的描述。

类是对象的模板，对象是类的实例。

算法的概念

算法是与具体语言的语法结构无关的解题思路。常用表示工具有程序流程图和N-S流程图。

算法的五个特性：

  • 有穷性：有限步骤、有限时间内结束；
  • 确定性：每一步无歧义，相同输入对应相同执行逻辑；
  • 可信性：所有步骤都可实际执行；
  • 输入：0个或多个合法输入；
  • 输出：至少一个结果输出。

程序质量评价

  • 空间复杂度：耗费计算机内存空间
  • 时间复杂度：耗费计算机的运行时间

Python 语言及其特点

Python 程序的特点

  1. 面向对象、解释型、弱类型的跨平台脚本语言
  2. 开源型编程语言：Python是自由/开放源码软件
    (Free/Libre and Open Source Software), FLOSS。
  3. Python 是一种功能强大的编程语言，具有大量的外部类库
  4. Python 具有极佳的可读性

Python 程序入门

  1. Python 运行环境

    1. 版本体系：2.x 和3.x， 彼此不互通
    2. 集成化编程环境：Python自带的IDLE

  2. IDLE中的两种运行方式

    1. 命令行交互方式：每次输入一条语句，按回车键后该语句被直接执行。此方式直接运行在IDLE的Shell主窗口中。

    2. 程序文件方式：把多个程序语句写在一个独立程序文件中，并把程序文件命名为以.py为拓展名的纯文本文件。

基于IDLE调试Python程序

程序调试入门

程序调试是为了发现程序中的错误并及时纠正。

  1. 语法错误的修正：利用系统提示修正语法错误。
  2. 逻辑错误的修正：以特定值是否符合预期来检验逻辑错误。

利用Python的调试环境调试程序

Python编程环境(IDLE)内置了程序调试功能(即Python Debug.pdb)。

调试功能简介：

  1. “Go”按钮：单机该按钮将导致程序正常执行至终止，或达到一个预设的断点。
  2. “Step”按钮：逐步调试按钮——严格逐步。单击该按钮将导致调试器执行下一个代码，然后再次停下来。若遇到函数，则会步入函数内部，继续做单步调试。
  3. “Over”按钮：逐步调试按钮——非严格逐步。把函数当作一条语句来执行，并不进入函数内部。
  4. “Out”按钮：结束当前函数。单击该按钮，将导致调试器全速执行当前函数内的代码行，直到他从当前函数中退出——与”Step”功能相辅相成，在Step进入某个函数中后，可借助Out快速完成此函数，以便返回上一层程序。
  5. “Quit”：立马终止该程序。
